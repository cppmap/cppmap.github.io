    C++ 標準ライブラリ¶

    C++ の規格で定められた標準ライブラリに含まれるヘッダの一覧です。
    リファレンス Web サイトは

    • en: cppreference.com,
    • jp: cppreference.com (日本語訳)
    • cpprefjp: cpprefjp

    に対応しています。
